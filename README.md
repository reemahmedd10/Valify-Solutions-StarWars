# Star Wars Dashboard

A React-based dashboard application that interacts with the Star Wars API (SWAPI) to display information about Star Wars characters, starships, and planets.

## Features

- **Characters**: Browse and search Star Wars characters
- **Starships**: Explore iconic vessels from the saga
- **Planets**: Discover diverse worlds across the galaxy
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Dark theme with Star Wars aesthetic

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **API**: SWAPI (Star Wars API)

## Getting Started

### Prerequisites

- Node.js 18.17.0 or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd star-wars-dashboard
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
